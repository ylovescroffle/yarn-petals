# Yarn Petals

A partnership proposal webpage for Yarn Petals - handcrafted yarn flower bouquets.

## About

This website presents a Year 1 growth plan and partnership proposal for Yarn Petals, a Melbourne-based business specializing in handmade yarn flower arrangements for graduations, weddings, and special occasions.

## Features

- 6 tabbed sections: Introduction, Vision, Timeline, Universities, Realistic Numbers, Let's Talk
- Responsive design for mobile and desktop
- Custom color scheme (coral, cream yellow, soft pink)
- Interactive navigation with smooth transitions
- Copy protection enabled

## Live Demo

Deployed on Vercel.

## Local Development

Simply open `index.html` in a browser - no build step required.

## Tech Stack

- HTML5
- CSS3 (with CSS variables)
- Vanilla JavaScript
